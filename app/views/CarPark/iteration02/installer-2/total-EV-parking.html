How many EV parking bays will the infrastructure support?

We want to know how many EVs the infrastructure will be able to charge, before you install any chargepoints.

The grant will pay up £500 for each EV the infrastructure let you charge, regardless of whether you install the chargepoints now or later.

To get the grant for infrastructure, it will need to charge at least 5 EVs at the same time and install 1 chargepoint.

What do we mean by infrastucture?
  • Cable routes, by which we mean channels for chargepoint cabling that run from the power supply to the chargepoint location
  • Electrical containment systems
  • Any associated reinstatement work,
  • Signage
  • Earthing arrangements
  • Any onsite one off software purchases, but not ongoing software support or services
  • Grid connections, but not grid reinforcement
  • Concrete plinths or footings for future chargepoints
  • Vehicle barriers
  • Upgrades to existing electrical infrastructure
  • Labour
  • Project management costs
